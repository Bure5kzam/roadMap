## 유저 

유저 구성은 우분투 보안관리의 기본입니다.

1. 파일 접근 권한 제어
2. 명령어 실행 권한 제어
3. 하드웨어 리소스 점유 권한 제어
4. 작업 내역, 접속 기록이 남음 (모니터링)
5. 계정 비밀번호 변경 주기 관리 및 비활성화를 구성할 수 있음.
